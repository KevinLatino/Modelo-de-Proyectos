Dale seguimiento a tu proyecto, siempre se puede mejorar

Una vez que ha concluido el proyecto, se debe realizar una fase de autocontrol con el fin de aprender a evaluar mejor la calidad del proyecto.
Para esta fase de autocontrol se suelen utilizar también cuestionarios o test programados similares a los que se suelen utilizar para los exámenes.
